typedef struct sxbp_co_ord_cache_t sxbp_co_ord_cache_t

Struct type for holding a cached set of co-ords.

This is useful for co-ords that are intended for re-use between operations which require co-ords which are certified as being correct up to a given line index.

typedef uint32_t sxbp_length_t

Type for representing the length of a line segment of a spiral.

Note
Although the width of this type is 32 bits, it is actually only 30 bits when used in the sxbp_spiral_t struct type. This is because here it is a bitfield field with 30 bits allocated to it.
typedef struct sxbp_line_t sxbp_line_t

Represents one line segment in the spiral structure.

This includes the direction of the line and it's length (initially set to 0).

Note
The whole struct uses bitfields to occupy 32 bits of memory.
typedef int8_t sxbp_rotation_t

Type for representing a rotational direction.

This can be SXBP_CLOCKWISE or SXBP_ANTI_CLOCKWISE.

Todo:

typedef struct sxbp_spiral_t sxbp_spiral_t

Struct type representing a Spiral figure, in any state of completion.

This is the most important data type in the whole library, and is passed to and from many different library functions. A partially-complete spiral object may be produced from binary data, and a fully-complete spiral may be produced from the partially complete one via the routines in the library.

Note
The solved_count and seconds_spent fields are currently unused by the solver/generator code, but they are read to and written from files. They will be used by the rest of the code in future versions.
typedef enum sxbp_status_t sxbp_status_t

Represents the success or failure status of an executed function.

Not all functions have this as their return type, but most do. Functions that may encounter error conditions which need to be propagated back to the caller will have this type as their return type.

typedef uint32_t sxbp_version_hash_t

Used for indexing and comparing different versions in order.

Versions that are behind a given version will compare as less than that version. Versions that are ahead will compare as greater than.

typedef struct sxbp_version_t sxbp_version_t

Used to represent a version of libsxbp.

Versions are of the format <MAJOR.MINOR.patch>
