sxbp_serialise_result_t sxbp_dump_spiral ( sxbp_spiral_t  spiral,
sxbp_buffer_t *  buffer 
)

Serialises a spiral to a buffer.

Writes out a binary representation of a given spiral to a buffer, which can then be stored in a file or sent across the network as required.

Parameters
spiralThe spiral which should be serialised to buffer.
[out]bufferThe data buffer to write out the spiral data to.
Returns
For information on return values, see the documentation of the return types.
Note
Asserts:
  • That spiral.lines is not NULL
  • That buffer->bytes is NULL
See also
sxbp_status_t for generic error return codes and sxbp_deserialise_diagnostic_t for file-specific error return codes.

The return type of this function could be changed to sxbp_status_t, as none of the serialisation-specific error codes can ever be returned by it.

De-serialises a spiral from a buffer.

Reads in a binary representation of a spiral and populates a given spiral with the data which represents this spiral (if input data is valid).

Parameters
bufferThe data buffer to load the spiral from.
[out]spiralThe spiral to write the spiral data to.
