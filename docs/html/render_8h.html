typedef struct sxbp_bitmap_t sxbp_bitmap_t

Used to represent a basic 1-bit, pure black/white bitmap image.

The image has integer height and width, and a 2-dimensional array of 1-bit pixels which are either black or white.

Renders the line of a spiral to an image format.

The lines of the spiral are plotted on a white background and the pixel data representing the resulting shape is written as data in a given image file format, using the callback function to achieve this conversion. The callback function should inspect the pixels of the bitmap passed to it and write the file data representing the bitmap in it's respective file format out to the buffer.

The function signature of the callback is the same as that of the image rendering functions provided by the library: sxbp_render_backend_pbm and sxbp_render_backend_png. This allows one of these to be provided as the callback function, a convenient use of function chaining can achieve this like so:

1 sxbp_render_spiral_image(spiral, buffer, sxbp_render_backend_pbm);
Parameters
spiralThe spiral which should be rendered.
buffer[out] The data buffer to which the bytes of the image file should be written.

Returns
SXBP_OPERATION_OK on success.
Any other valid value of the enumeration sxbp_status_t on error.
Note

  • That the function pointer is not NULL
